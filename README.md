# Inheritance UMaT Management System

## Description
This project demonstrates the concept of inheritance in Python by creating a simple UMaT Management System.

## Classes
- Person (Parent Class)
- Student (Child Class)
- Lecturer (Child Class)

## Features
- Stores common information (name and age).
- Student can enroll in a course.
- Lecturer can teach a course.
- Uses `super()` to initialize inherited attributes.
- Demonstrates method overriding using `display_info()` in the Student class.
